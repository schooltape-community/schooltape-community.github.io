New!

"Every Schoolbox needs Schooltape!"

Schooltape is a free, open-source, chrome extension that allows you to customise the look, feel, and functionality of Schoolbox!

It is currently in beta, so expect bugs and glitches. If you find any, please report them here.

Advanced Schoolbox Detection

Schooltape can detect if you are on a Schoolbox page, and will only run on those pages. This means that you can use Schooltape on any website, without it interfering with the page.

Schooltape will also not activate during quizzes or tests, so you can use it without worrying about it interfering with your work.

How to Install

Installing Schooltape is easy. Just follow the steps below:

  1. Download the latest version of Schooltape from the releases page.
  2. Unzip the file.
  3. Go to chrome://extensions in your browser.
  4. Enable developer mode.
  5. Click "Load unpacked" and select the folder that you unzipped. Navigate to the /Extension directory.
  6. Enjoy!
